Бан по логину

Смысл бана (запрещение доступа) по логину заключается в блокировании входа в чат с определенных логинов. Это актуально, например тогда, когда пользователь отказывается удалять из инфы порнографию, нецензурные статусы и в подобных случаях. На странице приведен список забаненых логинов. Если нажать на логин левой кнопкой мыши, то можно посмотреть информацию об данном логине. (Примечание: Эта информация доступна, только если Ваш уровень не ниже админа). Если требуется изменить сортировку списка, то следует нажать на имя того поля, по которому требуется отсортировать (имена полей написаны синеньким цветом).

Добавление логина

Добавить (забанить) логин можно двумя путями. Первый: по нику. Для этого нужно ввести в поле "Добавить по нику", требуемый ник. Второй способ: для него требуется ввести непосредственно логин в поле "Добавить по логину" требуемый логин. (Примечание: Вы можете забанить только те логины, у которых уровень ниже Вашего) Примеры:

Добавить по нику: Fox
Добавить по логину: mbr

Снятие бана
Снятие бана с логина делается очень просто. Для этого нужно нажать на против требуемого логина кнопку "Снять". Так же имейте ввиду, что Вы можете снимать баны с логинов, которые забанили либо Вы, либо любой другой, но кто по уровню ниже или равен Вашему. Если на против требуемого логина нет кнопки "Снять", то это означает что у Вас просто недостаточно прав, чтобы разблокировать данный адрес.

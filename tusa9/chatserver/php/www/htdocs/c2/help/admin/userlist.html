Спискок пользователей

Любимая страничка администраторов. Отсюда делаются почти все самые необходимые действия. Например одним кликом мышки можно запретить (забанить) вход какому-либо пользователю по IP, логину, нику. Так же только отсюда делается "кик в помойку", т.е. отправка пользователя в комнату "помойка" на 2 минуты (т.е. он не может сменить комнату 2 мин.)
Также список пользователей удобно использовать для самых разных целей, например для вычисления рядом ли сидят 2 человека.
Так же удобно пользоватся сортировкой, по IP, Proxy-IP, Логину, Нику.
При нажатии на можно посмотреть информацию об адресе, название сети, и пр. данные.

Будьте внимательны, любое действие на этой странице приводит к выкидыванию пользователя из чата.
Чтобы выставить пользователя из чата + забанить ник, достаточно ткнуть на ник.
Чтобы выставить пользователя из чата и забанить его логин на 5 минут, достаточно кликнуть на его логин. Аналогично с IP & Proxy-IP.
Так же существует возможность просто выкинуть пользователя из чата с этой странички. Для этого жмём на :)

Страничка "Список пользователей" автоматически обновляется раз в 2 минуты, но если требуется обновить скорее - то для этого есть кнопка "Обновить" внизу страницы.

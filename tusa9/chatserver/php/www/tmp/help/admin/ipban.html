Бан по IP

Смысл бана(запрещение доступа) по IP заключается в блокировании входа в чат с определенных IP адресов. Причем IP адреса может быть 2, если человек сидит через Proxy-сервер. Первый IP это реальный адрес пользователя (может быть любым, даже такой как "unknow") и второй IP адрес - адрес Proxy-сервера. Ниже приведены примеры:

IP: 192.168.0.1 Proxy: 212.192.117.9
IP: 217.106.120.3
IP: 192.168.0.19 Proxy: 212.192.117.9
IP: 192.168.0.19 Proxy: 217.106.125.15

Как видно из примера, IP адрес может быть одинаковый. Да! это может быть, но как видно из того же примера, у этих людей адреса Proxy-серверов разные. Если требуется изменить сортировку списка, то следует нажать на имя того поля, по которому требуется отсортировать (имена полей написаны синеньким цветом).

Добавление адреса

Чтобы добавить (заблокировать вход с адреса), надо в поле добавить адрес написать необходимый адрес. Это при условии что человек не использует прокси. Иначе дело обстоит немного сложнее. Если ввести в поле "адрес" реальный адрес человека, то все остальные, использующие прокси, и имеющие такой же внутренний адрес так же не смогут попасть в чат. Следовательно, если Вы хотите запретить доступ по адресу и прокси то требуется ввести в поле "Proxy" адрес Proxy-сервера. Примеры:

Адрес: 212.192.110.1
Адрес: 192.168.0.1
Адрес: 192.168.0.1 Proxy: 212.192.117.9

Снятие бана
Разблокирование адреса делается очень просто. Для этого нужно нажать на против требуемого адреса кнопку "Снять". Так же имейте ввиду, что Вы можете снимать баны с адресов, которые забанили либо Вы, либо любой другой, но кто по уровню ниже или равен Вашему. Если на против требуемого адреса нет кнопки "Снять", то это означает что у Вас просто недостаточно прав, чтобы разблокировать данный адрес.
